# Emoticons inline-bot for Telegram 

### wow! (╯°□°）╯︵ ┻━┻

##### Usage:
`@emotimsgbot [?param] [name/text]`,
 where `?param` can be:
  * `?l` - move text to left side
  * `?r` - move text to right side
  * `?n: [name]` - set name for emoticons with name
